# SAR Ship detection with Detection2

## Requirements
+ torch 1.6+cu101
+ detectron2 0.3
+ opencv
```bash
python -m pip install detectron2 -f \
  https://dl.fbaipublicfiles.com/detectron2/wheels/cu101/torch1.6/index.html
```
+ opencv
```bash
pip install opencv-python   
```

## Run
```bash
sh run.sh
```
